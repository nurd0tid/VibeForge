# Brand Assets

This document defines the rules for displaying VibeForge brand assets.

## Asset Files
- **Logo**: `public/logo.png`
- **Icon / Favicon**: `public/icon.png`

## Fallback
- If the image files are missing or fail to load, display the text **"V"** as a fallback.

## Color and Theme
- The icon must be readable and visually clear on both light and dark backgrounds.

## Rules
- Never use remote assets or placeholders (like external URLs) if local assets are available.
- Verify asset path resolution works correctly across routes.
